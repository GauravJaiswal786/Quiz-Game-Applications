A high-performance, dynamic quiz application built with *Vanilla JavaScript. This project demonstrates advanced **DOM Manipulation*, state management, and responsive UI design principles.

## 💎 Key Features
* *Dynamic Content Rendering:* Questions and options are injected into the DOM dynamically using JavaScript objects.
* *Real-time Scoring Logic:* Instant evaluation of user inputs with a final performance summary.
* *Responsive Architecture:* Fully optimized for Mobile, Tablet, and Desktop using *CSS Flexbox/Grid*.
* *State Management:* Efficiently tracks user progress, current question index, and total score during the session.
* *Smooth UX:* Integrated hover effects and transitions for an engaging user experience.

## 🛠️ Technical Stack
* *Frontend:* HTML5, CSS3 (Custom animations & transitions)
* *Core Logic:* JavaScript (ES6+)
* *Data Structure:* Array of Objects (JSON format style)

## 📖 Technical Implementation (For Interviewers)
1. *Event Driven:* Used high-performance *Event Listeners* to capture user interaction.
2. *Conditional Logic:* Implemented complex if-else and switch logic for answer validation and score calculation.
3. *Template Literals:* Used ES6 Template Literals for clean and readable HTML injection via JavaScript.
4. *Logic Flow:* Designed a seamless flow from the landing screen to the final results page without manual refresh.
